C is a structured programming language. Developed by Dennis Ritchie in Bell Laboratories in 1972.
C is a general-purpose programming language.
C is strongly associated with UNIX, as it was developed to write the UNIX operating system.

Why learn C ?

1. It is one of the most popular programming languages in the world
2. If you know C, you will have no problem learning other popular programming languages such as Java, Python, 
   C++, C#, etc, as the syntax is similar
3. C is very fast, compared to other programming languages, like Java and Python
4. C is very versatile; it can be used in both applications and technologies.

Difference between C and C++

C++ was developed as an extension of C, and both languages have almost the same syntax
