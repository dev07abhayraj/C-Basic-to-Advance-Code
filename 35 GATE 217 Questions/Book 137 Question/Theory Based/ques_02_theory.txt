---------------------------------------------- Question No. 02 -------------------------------------------------

Year  : 1989

Marks : 2

Theory type :

In which of the case(s) is it possible to obtain different results for call-by-reference and call-by-name parameter passing ?

(a). Passing an expression as a parameter
(b). Passing an array as a parameter
(c). Passing a pointer as a parameter
(d). Passing an array element as a parameter

---------------------------------------------- ANSWER -------------------------------------------------

Answer is (d) Passing an array element as a parameter
